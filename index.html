Screen obyekti

Javascriptdə istifadəçinin ekranı haqqında məlumatları özündə saxlayan screen obyekti mövcuddur. Bu obyekt bir sıra xüsusiyyətlərdən ibarətdir.
width və height xüsusiyyətləri — istifadəçinin ekranının enini və uzunluğunu bizə piksellə verməkdədir. 

                document.write("En: " + screen.width);
              
                document.write("Uzunluq: " + screen.height);

Beləliklə ekranın enini və uzunluğunu çap etmiş oluruq.

availWidth və availHeight xüsusiyyətləri — Windows Taskbar istisna olmaqla ekranın enini və uzunluğunu göstərməkdədir. 

            document.write("En: " + screen.availWidth);
            
            document.write("Uzunluq: " + screen.availHeight);

Beləliklə, məlumatları çap etmiş oluruq.
